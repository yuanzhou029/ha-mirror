# HA Mirror
Home Assistant 完整镜像仓库，包含官方后端核心和前端。
## 项目结构
ha-mirror/
|-- backend/          # Home Assistant 后端核心
|-- frontend/         # Home Assistant 前端
|-- scripts/          # 构建和部署脚本
|-- docs/             # 文档
|-- .github/          # GitHub Actions 工作流
|-- README.md
## 功能特性
- 官方 Home Assistant 后端核心
- 官方 Home Assistant 前端
- 支持 GitHub Packages 作为 PyPI 仓库
- 自动构建和发布流程
- 版本控制和更新管理
